# Platzigram

An Instagram clon: SPA 

## Description

Building most important features of instragram front end using several tools as:

NodeJS, ExpressJS, Browserify, Gulp, PageJS, Yo-yo, Jade, MaterializeCss, Sass, BabelJS, Amongst others.

functionalities:

1. Home page
2. User page
3. Picture detail
3. Load Image
4. Like picture on the picture frame
5. Like picture on the picture card frame
6. Loading effect
7. Biligual site
8. Full Responsive
